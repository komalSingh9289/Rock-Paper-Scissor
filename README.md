# Rock-Paper-Scissor
Rock-Paper-Scissor Game Using HTML, CSS and JS.

How to play?
<br>
Step 1.  Click `Let's play` button. 
<br>
Step 2. Choose one of the options (rock,paper,scissor).
<br>
Step 3. Beat the Computer.
